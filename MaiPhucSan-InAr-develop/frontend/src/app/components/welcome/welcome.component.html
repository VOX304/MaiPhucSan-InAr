Welcome

This cockpit is a prototype for the semester project HighPerformance. It uses:

  • Node.js + Express REST backend
  • MongoDB persistence
  • Optional integrations: OrangeHRM, OpenCRX, Odoo
  • Optional workflow: Camunda

Quick demo steps

  1. Login as hr and create social performance records for a salesman (E1001).
  2. (Optional) refresh orders from OpenCRX.
  3. Login as ceo, compute bonus and approve.
  4. Login as hr, approve HR (stores in OrangeHRM) and release to salesman.
  5. Login as sales1, view and confirm the bonus.

API documentation is available at /api-docs on the backend.
